# <img src="https://icongr.am/fontawesome/magic.svg?size=26&color=bd93f9" align="center" alt="maquiagem"/> MAKEUP - Landing Page

> **Resumo:** Landing Page institucional desenvolvida para o segmento de Beleza e Estética. O foco do projeto foi traduzir a identidade visual da marca em uma interface elegante, com layout limpo

## <img src="https://icongr.am/feather/info.svg?size=30&color=bd93f9" align="center" alt="sobre" /> Sobre o Projeto
Esse projeto foi desenvolvido com o intuito de estrurar uma landing page fictícia, porém muito próxima da realidade, para uma empresa de maquiagens e cosméticos. Foram ultilizados conceitos de identidade visual e atratividade, além de respeitar a hierarquia das informações na página.


## <img src="https://icongr.am/feather/cpu.svg?size=26&color=bd93f9" align="center" alt="icon" /> Tecnologias Utilizadas
* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização avançada, uso de displays, interações e animações sutis.
* **Flexbox & Grid:** Para o layout e alinhamento dos elementos.

## <img src="https://icongr.am/feather/trending-up.svg?size=30&color=bd93f9" align="center" alt="evoluindo" /> Destaques Técnicos
* **Personalidade:** O layout se ajusta à proposta visual de muitas empresas reais que atuam no ramo da beleza.
* **Forms:** Intuitivo e elegante.
